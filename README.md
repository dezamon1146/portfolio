# Leopoldo C. Servaña III — Portfolio

Personal portfolio website showcasing my projects, skills, and background as a BSIT student specializing in web development.

## About
A dark-themed developer portfolio built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Features a projects showcase with filter by category, a working contact form via Formspree, and a smooth single-page layout.

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Share Tech Mono, Syne, Inter)
- Formspree (contact form)
- GitHub Pages (hosting)

## Features
- Responsive design (mobile & desktop)
- Animated hero with typewriter effect
- Projects grid with filter (web / mobile / software)
- Working contact form with no-redirect submission
- Download links for APKs and software releases
